# PROYECTO-ENTORNO-Y-LENGUAJE
# RED_SHIFT

Este proyecto lo hice en Visual Studio y su objetivo es crear una web sencilla para representar nuestra web. La idea principal es tener un lugar donde se puedan, consultar y sea fácil de usar.

## ¿Qué aprendí o qué trabajé aquí?

A lo largo del proyecto pude:

- Poner en práctica lo que he aprendido en Visual Studio.  
- Trabajar con la estructura de un proyecto real y organizar bien los archivos.  
- Crear una interfaz que sea clara y que cualquiera pueda usar sin muchas vueltas.  
- Manejar datos
- Entender mejor cómo funciona una web desde que se abre hasta que se usa cada función.

---

## Conclusión

Este proyecto no solo es una web para promocionarnos , sino también una forma de aplicar mis conocimientos, mejorar mi lógica de programación y entregar algo útil y práctico. Es una buena experiencia para seguir aprendiendo y avanzar en mis habilidades como desarrollador.
